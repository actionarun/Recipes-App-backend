# Recipe CRUD API

A RESTful CRUD API for managing recipes built with Node.js, Express.js, and MongoDB using the MVC architecture.

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman

## Features

- Create a new recipe
- Get all recipes
- Get recipe by ID
- Update recipe by ID
- Delete recipe by ID
